# CRM das Joias 💍

Este projeto é um **CRM (Customer Relationship Management)** desenvolvido para gestão de clientes, produtos e vendas numa loja de joias.

O objetivo principal é **aprender e consolidar competências de desenvolvimento**, aplicando conceitos práticos num projeto realista.

---

## 🚀 Funcionalidades (em evolução)

- Gestão de clientes
- Registo e organização de joias/produtos
- Controlo de vendas
- Histórico de clientes
- Estrutura pensada para expansão futura

---

## 🛠️ Tecnologias Utilizadas

- C# , .Net, REACT ,  Javascript.
- DOCKER, NGINX, VPS
- SQL
- Git e GitHub para controlo de versões

> Nota: As tecnologias podem evoluir ao longo do desenvolvimento.

---

## 📚 Objetivos do Projeto

- Praticar desenvolvimento de aplicações
- Melhorar lógica de programação
- Aplicar boas práticas de código
- Trabalhar com Git em ambiente real
- Criar um projeto portfolio-ready
